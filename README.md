# smalljs

## Dev

```bash
cd /path/to/mam && npm start
# Open http://localhost:9080/bog/smalljs/app/-/test.html
```

## Build

```bash
npx mam bog/smalljs
```

## Deploy

Push to `main` → GitHub Actions → GitHub Pages: https://bog.github.io/smalljs/

Feature branches deploy to: https://bog.github.io/smalljs/{branch-name}/
